# GREATMOON MOTOR SPARES

A professional website for GREATMOON MOTOR SPARES, a motor spares business based in Mutare, Zimbabwe.

## Features

- Responsive design that works on mobile, tablet, and desktop
- Product categories showcase
- WhatsApp integration for easy customer inquiries
- Contact information and business hours
- About page with company information
- Google Maps integration

## Technologies Used

- HTML5
- CSS3 (with CSS variables for easy customization)
- JavaScript (vanilla, no frameworks)
- Vercel for hosting

## Deployment

This site is configured for easy deployment on Vercel. To deploy:

1. Push this code to a GitHub repository
2. Go to [vercel.com](https://vercel.com) and sign up/login with GitHub
3. Click "Add New Project" and import your GitHub repository
4. Vercel will automatically detect the static files and deploy
5. Your site will be live at a URL like `your-project.vercel.app`

## Customization

### Update Business Information

Edit the following files to update your business details:

- `script.js` - Update the contact constants at the bottom of the file
- `contact.html` - Update the Google Maps embed with your actual location
- All HTML files - Update any text content as needed

### Colors

The color scheme can be customized by editing the CSS variables in the `:root` section of `styles.css`.
